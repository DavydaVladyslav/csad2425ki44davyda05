void resetValues ( )

Resets the game values to their initial state. This function performs the following actions:

  • Sets the game_started flag to "0".
  • Clears the message string. -Initializes the board array with the value "_" Note: Some variables such as connect_arduino, game_mode, and ai_strategy are commented out and not reset by this function.
